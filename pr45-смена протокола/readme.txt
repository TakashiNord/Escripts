
так как у Вас 1 прибор = 1 Профиль, то задача немного облегчается. 
Нет необходимости делать копию профиля, а потом изменять привязку параметров и квалификаторов.
---------------------------------------------------

Шаг1. Подготовительный.
  [Навигатор БД]
    запишите на листе id Порта и id Прибора,
    также , резервного id (Порта и id Прибора)

Шаг2. Запустите 
  [Справочная информация по распределенной системе сбора]
  Справочники - > Устройства сбора.

  добавьте для Устройства сбора по МЭК101 - протокол МЭК104

Шаг3. Открываете Toad.
 1) таблица da_dev
  нам необходимо узнать id - Устройства сбора

записываете его iddev
  
 2) таблица da_dev_desc
    фильтруете записи по полю ID_DEV = iddev

 3) таблица da_dev_desc 
   Самый главный этап. Нам необходимо сменить структуру МЭК101 на МЭК104
   это поле ID_PROTO_POINT

согласно Вашей таблице DA_PROTO_DESC
32      	IEC60870-5-101	        456
33	32	Общий адрес ASDU	457
58	33	Группа	457
34	58	Тип	457
35	34	Точка	458


48		IEC60870-5-104	456
49	48	Общий адрес ASDU	457
59	49	Группа	457
50	59	Тип	457
51	50	Точка	458


значит, изменения будут такими:

update DA_PROTO_DESC set ID_PROTO_POINT=48 where ID_PROTO_POINT=32 and ID_DEV=iddev ;

update DA_PROTO_DESC set ID_PROTO_POINT=49 where ID_PROTO_POINT=33 and ID_DEV=iddev ;

update DA_PROTO_DESC set ID_PROTO_POINT=59 where ID_PROTO_POINT=58 and ID_DEV=iddev ;

update DA_PROTO_DESC set ID_PROTO_POINT=50 where ID_PROTO_POINT=34 and ID_DEV=iddev ;

update DA_PROTO_DESC set ID_PROTO_POINT=51 where ID_PROTO_POINT=35 and ID_DEV=iddev ;


commit ;


все. мы изменили Профиль с МЭК101 на МЭК104.

 5) Теперь, нам необходимо сменить протокол на Приборе и Портах.
    таблица DA_PORT
    поле ID_NODE  - это id порта\прибора
    поле ID_PROTO - это id протокола  (можете отфильтровать по ID_PROTO = 32   - МЭК101)
    
 у нужных  ID_NODE (которые мы записали на шаге1 ) меняете на ID_PROTO = 48  ( МЭК104 )

 !!!!!! коммитите

  6) Запускаете [Навигатор БД]
     проверяете свойства на порту и приборах. Резервном.
     При нажатии на кнопке Профиль - должен быть переход на наш измененный профиль.

     Для Приборов необходимо проверить заданные свойства. Если - нет нужных, для МЭК104, задать.

    запускаете [Справочная информация по распределенной системе сбора] можете 
 переименовать профиль как МЭК 104.
